PBI Framer gap upgrade patch

Adds:
- Freeform object positioning in Visual Studio with X/Y/width/rotation/z controls.
- Multi-page canvas editing with page switcher, add, duplicate and delete pages.
- Advanced animations: float, reveal, parallax, stagger, marquee, plus duration/delay/easing controls.
- CMS database UI with cloud save/load and D1-backed endpoint.
- Collaboration panel with invites, notes, presence, same-device BroadcastChannel sync and D1-backed endpoints.
- Published renderer support for multi-page canvas output and freeform layers.

Note: real Google Docs-style simultaneous multiplayer editing would need a durable realtime layer such as Cloudflare Durable Objects/WebSockets. This patch adds the product UI, storage endpoints, presence and sync scaffold that can be upgraded into true realtime locking later.
