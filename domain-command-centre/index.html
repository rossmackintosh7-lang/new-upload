Domain & email

Connect the website without turning DNS into soup.

A practical command centre for domain connection, email setup, DNS checks, Stripe/Google verification and launch readiness.

Domain connection checklist

  • Choose PBI subdomain, connect existing domain or request a new domain.
  • Confirm who controls the registrar account.
  • Check DNS records before changing anything.
  • Add website records only after email records are backed up.
  • Confirm SSL/HTTPS after DNS changes settle.

Email setup checklist

  • Decide if the business needs info@, hello@ or support@.
  • Choose mailbox provider or forwarding route.
  • Set MX, SPF, DKIM and DMARC records.
  • Check Resend sender records if PBI sends website emails.
  • Send a test enquiry before launch.
