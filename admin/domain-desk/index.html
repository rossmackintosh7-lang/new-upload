Admin subdomain setup

You do not need to buy a new domain. Add admin.purbeckbusinessinnovations.co.uk as a subdomain/custom domain in Cloudflare.

  1. Workers & Pages → new-upload → Custom domains.
  2. Add admin.purbeckbusinessinnovations.co.uk.
  3. Point it to the same Pages project.
  4. Keep /admin/ protected by admin login.

Customer domain checklist

  • Confirm root redirects to www.
  • Check SSL active.
  • Check enquiry form sends.
  • Check email DNS before changing site records.
  • Add notes to the customer project.
