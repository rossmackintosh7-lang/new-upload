What the canvas now does:
- Component palette with draggable blocks.
- Drag blocks into the canvas.
- Click blocks into the canvas.
- Drag blocks to reorder.
- Move blocks up/down.
- Duplicate blocks.
- Delete blocks.
- Inline edit titles and text directly on the canvas.
- Inspector edit panel for:
  - title
  - text
  - button text
  - image URL
  - layout
  - background colour
  - accent colour
  - padding
  - alignment
- Responsive preview modes:
  - desktop
  - tablet
  - mobile
- Undo and redo.
- Autosave to local storage.
- Local versions.
- Restore local versions.
- AI build from brief.
- Safe local AI rewrite helpers:
  - clearer
  - SEO focus
  - shorter
- Export/apply the canvas back into the normal PBI builder.

Important honesty:
- This is a real browser-based drag-and-drop canvas.
- It is not yet a full commercial Framer/Webflow-grade layout engine with absolute positioning, nested containers, breakpoints per element and collaborative editing.
- It is designed to work safely with PBI's existing builder/save/publish flow by exporting the canvas into current project content.
- The next upgrade after this would be true nested containers, section-specific save format in D1, cloud canvas sync and published-site rendering directly from canvas JSON.
