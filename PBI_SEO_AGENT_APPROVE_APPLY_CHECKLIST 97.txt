PBI SEO AGENT APPROVE & APPLY UPGRADE

What this adds:
- Fix still creates an SEO suggestion.
- Approve & apply now writes a live SEO override into D1.
- A Cloudflare Pages middleware applies approved fixes to served HTML.
- The source files are not rewritten. This is safer for launch.
- Run Scan again after approval to confirm issues are resolved.

Approved fix types currently applied:
- Title
- Meta description
- H1
- Canonical
- Robots
- JSON-LD schema
- Missing image alt text
- Internal link block
- Thin content/content expansion block

How it works:
1. SEO Agent scan finds issues.
2. Click Fix on an open issue.
3. Review the generated suggestion.
4. Click Approve & apply.
5. The suggestion status becomes applied.
6. A row is added/updated in seo_page_overrides.
7. functions/_middleware.js applies the override when the public page is served.
8. Run Scan again to check the result.

Required after upload:
npx wrangler d1 execute d1-template-database --file=./database/pbi-combined-schema.sql --remote

Important:
This does not commit source-code changes back to GitHub.
For permanent source-file changes, manually copy approved improvements into the HTML later.
