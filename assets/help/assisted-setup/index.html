PBI support guide

Assisted Setup helps turn a draft into a launch-ready website.

Use Assisted Setup when you want help with structure, wording, images, settings and final launch checks.

What Assisted Setup is for

Assisted Setup is a one-off support option for customers who can start the website themselves but want PBI to help make it cleaner, clearer and more ready to launch.

It can include help with

  • Improving homepage wording and service descriptions.
  • Choosing sensible pages for the business.
  • Checking layout, spacing and mobile view.
  • Reviewing images and where they should sit.
  • Checking contact buttons, enquiry routes and calls-to-action.
  • Helping with domain and publishing settings.
  • Explaining what still needs to be supplied by the customer.

What the customer should prepare

Assisted Setup works best when the customer provides real business details: services, prices if relevant, opening hours, contact details, service area, images and any existing website or social links.

What it is not

It is not a promise of sales, rankings or instant results. It is practical setup help so the website feels clearer, more professional and easier for customers to use.
